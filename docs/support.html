RemoteCTRL Support

Last updated: June 28, 2026

Quick Start

  1. Install the RemoteCTRL server on your machine
  2. The server generates a Cloudflare tunnel URL
  3. Open RemoteCTRL on your iPhone and enter the URL
  4. Start managing your server remotely

Common Issues

Cannot connect to server

  • Ensure the RemoteCTRL server is running on your machine
  • Check that the tunnel URL is correct (Settings → Connection)
  • Verify the API token matches between server and app
  • Try the "Test" button in Settings → Connection

Terminal is slow or unresponsive

  • Check your network connection
  • Try closing unused sessions (they consume server resources)
  • Restart the server if the issue persists

File upload fails

  • Ensure you have write permissions on the target directory
  • Check available disk space on the server
  • Try smaller files first to verify the connection

Push notifications not working

  • Ensure notifications are enabled in Settings → Push Notifications
  • Check iOS Settings → RemoteCTRL → Notifications is enabled
  • Restart the app after enabling notifications
