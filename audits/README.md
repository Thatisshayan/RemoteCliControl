# Private Audit Vault

Use this folder for audit files that should stay in the repo workspace but should not be committed publicly by default.

## Naming Rule

Audit filenames must follow:

`DD.MM.YYYY<AgentName><Scope>Audit.md`

Example:

`17.07.2026CodexGeneralAudit.md`

## Rules

- save private audit working files here
- do not delete audit files without direct approval from Shayan
- promote an audit into `docs/` only when it is intentionally meant to be public
- if an audit is promoted into `docs/`, note that decision in the report and commit
